# Docker Stack: PHP-FPM + Nginx + MariaDB

Une infrastructure blockchain développée en **PHP** pour garantir la précision des calculs financiers sans erreurs de nombre à virgule. Gestion des blocs, synchronisation des portefeuilles, traitement des transactions et validation dans un ordre d'initialisation précis. Architecture modulaire conçue pour évolution progressive du système.

## Démarrage

```bash
docker-compose up -d
```

## Accès

- http://localhost:81
